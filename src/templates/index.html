Bedienungsanleitung — Collage Tool

1. Layout wählen

Wähle eines der vordefinierten Raster (z.B. 2×1, 1×2, 2×2) oder klicke auf Benutzerdefiniert für ein beliebiges Zeilen/Spalten-Raster.

2. Canvas-Größe

Wähle eine Ausgabe-Auflösung aus dem Dropdown. Mit Benutzerdefiniert… kannst du eine beliebige Pixel-Größe eingeben.

3. Bild in ein Feld laden

Klicke auf ein graues Feld auf dem Canvas — ein Datei-Dialog öffnet sich. Wähle ein Bild (PNG, JPG, WebP). Das Bild wird automatisch auf das Feld skaliert (Cover-Modus: das Feld wird vollständig ausgefüllt).

4. Bild innerhalb des Felds verschieben & zoomen

  • Bild ziehen: Klick & Drag auf ein gefülltes Feld
  • Mausrad: Rein- und rauszoomen innerhalb des Felds
  • Doppelklick: Bild auf Cover-Größe zurücksetzen

5. Einstellungen

  • Bildtrenner: Abstand zwischen den Bildern in Pixeln
  • Hintergrund: Farbe zwischen den Bildern (Bildtrenner-Farbe)
  • Außenrahmen: Farbiger Rahmen um die gesamte Collage (Farbe + Breite in px)
  • Abrunden: Abgerundete Ecken für alle Felder

Alle Einstellungen wirken sofort auf den Canvas.

6. Export

Klicke auf Download PNG oder Download JPG — die Collage wird in der gewählten Ausgabe-Auflösung exportiert (off-screen, sodass der sichtbare Canvas unverändert bleibt).

7. Alles leeren

Entfernt alle geladenen Bilder. Layout und Einstellungen bleiben erhalten.
